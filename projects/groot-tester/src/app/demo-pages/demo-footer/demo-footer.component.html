Footer

You can create a footer by using the component groot-footer. You generally will want to add this after your div.container element, in your app's main component. By default the footer will show to the left its content, for which you might want to use the class logo, and to the right the language selector, which will show italian and english by default.

By default the footer will use the footer-sticky class, which will anchor it to the bottom. You can pass whatever classes you need to it, though, via the classes property. Note that if you use the footer, you need to add to your main container-fluid (the one that will contain the router-outlet) the class container-with-sticky-footer to avoid it sliding "under" the footer.

Example

<groot-footer>
  <div class="logo">
    
    List &copy; 2019
  </div>
</groot-footer>
