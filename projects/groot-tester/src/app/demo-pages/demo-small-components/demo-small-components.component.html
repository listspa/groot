Other small components

This page shows the various additional components that are present in Groot.

Yes - no check

True
<groot-yes-no-check [value]="true"></groot-yes-no-check>
False
<groot-yes-no-check [value]="false"></groot-yes-no-check>

Operations button

Action buttons with title and content
<groot-actions-button popoverTitle="the title">
  <groot-actions-button-entry label="test" (click)="onClickActionProva()"></groot-actions-button-entry>
  <groot-actions-button-entry label="dummy"></groot-actions-button-entry>
  <groot-actions-button-entry label="off" [disabled]="true"></groot-actions-button-entry>
</groot-actions-button>
Action buttons without title and with template

Actions:

 
<groot-actions-button [popoverTemplate]="actionsTpl"></groot-actions-button>

<ng-template #actionsTpl>
  <h2>Actions:</h2>
  <groot-actions-button-entry label="test" (click)="onClickActionProva()"></groot-actions-button-entry>
</ng-template>

Display labels

To show a simple table-like formatting of fields, you can use the simple groot-display-label-value. You can also use the "value" part with the component groot-display-value, which might be useful for tables' cells.

String formatting
<groot-display-label-value
  label="label"
  [value]="'a string'"
></groot-display-label-value>
Integers
<groot-display-label-value
  label="number"
  [value]="93128"
  [dataType]="NbpuSchemaFieldType.INT32"
></groot-display-label-value>
Double
<groot-display-label-value
  label="number"
  [value]="32312.4115"
  [dataType]="NbpuSchemaFieldType.DOUBLE"
></groot-display-label-value>
Double customizing number of decimals
<groot-display-label-value
  label="number"
  [value]="32312.4115"
  [dataType]="NbpuSchemaFieldType.DOUBLE"
  doubleFormat="0.4"
></groot-display-label-value>
Dates
<groot-display-label-value
  label="date"
  [value]="now"
  [dataType]="NbpuSchemaFieldType.DATE"
></groot-display-label-value>
Dates customizing format
<groot-display-label-value
  label="date"
  [value]="now"
  [dataType]="NbpuSchemaFieldType.DATE"
  dateFormat="dd.MMM.yyyy"
></groot-display-label-value>
Hour
<groot-display-label-value
  label="hour"
  [value]="now"
  [dataType]="NbpuSchemaFieldType.TIME"
></groot-display-label-value>
Timestamps
<groot-display-label-value
  label="timestamp"
  [value]="now"
  [dataType]="NbpuSchemaFieldType.TIMESTAMP"
></groot-display-label-value>
Boolean true
<groot-display-label-value
  label="boolean"
  [value]="true"
  [dataType]="NbpuSchemaFieldType.BOOLEAN"
></groot-display-label-value>
Boolean false
<groot-display-label-value
  label="boolean"
  [value]="false"
  [dataType]="NbpuSchemaFieldType.BOOLEAN"
></groot-display-label-value>
Multiple values
<groot-display-label-value
  label="multiple strings"
  [value]="['first', 'second', 'third']"
></groot-display-label-value>
Empty value
<groot-display-label-value label="empty"></groot-display-label-value>

Breadcrumbs

To show breadcrumbs in the top-left corner of the page, we have some components that follow Darwin (XDCE) style. Sample:

Go back link
<groot-darwin-breadcrumbs>
  <groot-darwin-breadcrumb-go-back
    [action]="['/']"
    label="home"
  ></groot-darwin-breadcrumb-go-back>
</groot-darwin-breadcrumbs>
