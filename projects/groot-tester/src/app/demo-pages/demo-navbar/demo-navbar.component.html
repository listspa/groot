Nav Bar

The menu bar on top of the application is called the "nav bar" in Bootstrap, and we have retained the name for Groot. To use it you need to do two things:

  1. in your root component (generally AppComponent) add the appropriate tag: 
    <groot-nav-bar [menu]="menu" rootMenuLabel="MyAppName"</groot-loading-indicator>
  2. in the relative class, define the menu variable, of type Menu[][] with your menu items.

The menu is an array of arrays; the first level is the columns (the navbar shows four columns), and the second level is the rows inside a given column. Each item must have a label and should have either an url, or a set of children (again a Menu[][] with the same logic). Each item might optionally have an icon.

The nav bar shows in the first line the "breadcrums", i.e. the currently selected submenu path. If you only have one level of menu items this is overkill; it can be disabled by passing [showAlwaysBreadcrumbs]="false" to the nav bar component.

Example

readonly menu: Menu[][] = [
  [
    {label: 'Home', icon: 'fa fa-home', url: '/home'}
  ],
  [
    {label: 'Docs', icon: 'fa fa-book', requiredCapability: 'see-docs', children: [
      [
        {label: 'Getting started', url: '/docs/getting-started', icon: 'fa fa-star-o'},
        {label: 'Getting help', url: '/docs/getting-help', icon: 'fa fa-ambulance'},
      ],
      [
        {label: 'Stylesheets and skins', url: '/docs/groot-style', icon: 'fa fa-paint-brush'},
        {label: 'Icons', url: '/docs/icons', icon: 'fa fa-picture-o'},
        {label: 'Translations', url: '/docs/translations-guide', icon: 'fa fa-plane'},
      ],
      [
        {label: 'Internet explorer', url: '/docs/ie-support', icon: 'fa fa-internet-explorer'},
      ],
      [
        {label: 'External references', url: '/docs/external-references', icon: 'fa fa-external-link'},
      ]
    ]}
  ]
]
