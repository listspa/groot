List of buttons

When you have a (say) search form, you often need to align multiple buttons to the right of it and leave some space between then. This can be done via our class buttons-list:

If you want to align buttons to the left, you can use the classes buttons-list or buttons-list-bottom-left. The first does not impose any vertical alignment, which is useful for stand-alone rows of buttons; the latter aligns the buttons to the bottom.
