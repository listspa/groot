Buttons

We have a very simple groot-button component that you can use, but for heavy customization just use the standard Bootstrap one.

Code sample:

<groot-button
  [classes]="btn-info"
  icon="fa fa-search"
  label="Info button"
  (click)="clicked()"
></groot-button>

When you have a (say) search form, you often need to align multiple buttons to the right of it and leave some space between then. This can be done via one of our classes: buttons-list-left, buttons-list-right, buttons-list-bottom-left or buttons-list-bottom-right:

Split-buttons

Sometimes it's useful to present a button that hides a menu of actions. To do so, we have created the component groot-split-button:
