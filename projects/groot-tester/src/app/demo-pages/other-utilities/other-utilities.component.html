Other utilities

Other than the various components, we have a few other utilities in Groot, detailed here.

Dates functions

The function isoDate formats a JavaScript Date object to a string in the ISO format (yyyy-mm-dd). The symmetric function parseDate can parse a string and returns the Date object.

Strings functions

leftPad and rightPad

The trivial functions leftPad and rightPad can pad a string by adding required characters to the left or to the right.

Browser detection

The functions isInternetExplorer11, isEdge and isIe11OrEdge can be used to detect whether we are running under those crappy browsers, in which case we might have to disable some functions.

Fast Track dates and times

To format FastTrack's date and times (SKLDate and SKLTime), we have two pipes, called ftDate and ftTime to allow for simple formatting.

Groot Date

To format any string containing a date in a specific format, we have Groot Date pipe. You can specify both input and output formats. For now, the pipe accept the following formats:

  • yyyy-MM-dd *default*
  • dd-MMM-yy
  • dd-MMM-yyyy

For the output format, you can use any standard Date pipe format. As a last parameter, you can specify the locale (default: IT).

Uploading files

To send a file to the server, the service UploadFileService can help. It offers a way to send a file, plus additionally various other paremters, in a format that works well with Spring. On the server you want to declare a @PostMapping method with a @RequestParam("file") MultipartFile file parameter.

Sorting arrays

It's pretty common to have an array of objects and to need to sort them by a property. You can use the helper function compareBy to help with that.

Pagination

The functions startIndex and endIndex help working with a Pagination object.
