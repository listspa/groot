Cards

Bootstrap supports the cards to wrap some content, such as a form. It looks pretty good and can be useful if you have a page with a couple of forms. We do not have a specific component at the moment, but rather simply a css customization.

The header by default looks the same as a groot-collapsible-box. However you can customize its color by using the classes card-header-primary (and similar).
