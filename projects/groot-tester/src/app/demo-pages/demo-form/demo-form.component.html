Forms

We have created some simple groot components to simplify the most common usages. In case you need to do something more advanced, such as various validations, you can take a look at the Angular form validation guide and at the Bootstrap documentation for forms.

Groot Components

Input box

Simple mandatory input field
<groot-input
        label="Name 0"
        name="name 0"
        [required]="true"
        [(ngModel)]="form.name"
      ></groot-input>
Help text and icon right
<groot-input
        label="Name 1"
        name="name 1"
        [required]="true"
        [(ngModel)]="form.name"
        iconRight="fa fa-user-o"
      ></groot-input>
Not mandatory field, help text and icon left
<groot-input
      label="Name 2"
      name="name2"
      [required]="false"
      [(ngModel)]="form.name"
      helpText="your first name, not your last name!"
      iconLeft="fa fa-image"
    ></groot-input>

Radio

Radio examples: selected, not selected, disabled

      <groot-radio name="groot-radio"

      <groot-radio name="groot-radio"

Textarea

Sample required textarea

Combo boxes

Combo box with string values

      <groot-combo
        label="Currency"
        name="currency"
        [items]="currencies"
        [required]="true"
        [(ngModel)]="form.currency"
      ></groot-combo>
Combo box with multiple selection

      <groot-combo
        label="Currency"
        name="currencyMultiple"
        [items]="currencies"
        [required]="true"
        [(ngModel)]="form.selectedCurrencies"
        [multiple]="true"
      ></groot-combo>
Combo box with object values

      <groot-combo
        label="Album"
        name="album"
        [items]="albums"
        [(ngModel)]="form.selectedAlbumId"
        bindLabel="name"
        bindValue="id"
      ></groot-combo>
Combo box with checkboxes

      <groot-combo-checkbox
        label="Currency"
        name="manyCurrency"
        [items]="manyCurrencies"
        [required]="true"
        [(ngModel)]="form.selectedManyCurrencies"
      ></groot-combo-checkbox>

Date picker

The ngModel maps to a standard JavaScript Date object.

Not mandatory field, with help text and icon left

      <groot-date-picker
        label="Date"
        name="date"
        [required]="true"
        [(ngModel)]="form.date"
        helpText="select a date please"
      ></groot-date-picker>
