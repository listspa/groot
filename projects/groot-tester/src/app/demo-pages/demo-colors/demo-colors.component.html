Colours and typography

Colours

Bootstrap includes various classes to highlight page components. We use (mostly) the standard classes, plus a couple of additional ones. The class name is always text-XXX and bg-XXX. For instance, we have text-primary and bg-primary.

Typography

We have customized the font; we use by default Roboto for the body and Roboto Condensed for the headings. You can of course change the fonts; take a look at the stylesheets and skins age. We include the fonts using WebFonts; you can take a look at the fonts.scss file to see how we have done it, in case you want to use another custom font.

For titles, use simply <h1> and similar tags.
