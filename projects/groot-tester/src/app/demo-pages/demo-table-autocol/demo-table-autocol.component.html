Tables with columns selector

In addition to the standard groot-table component, we also have a less customizable component, which automatically handles columns selection, drag to reorder and resizing. Here's an example:

{{value}}
{{row.mFamily}}-{{row.mGroup}}-{{row.mType}}

The component is called groot-table-autocol. To use it, you need to pass to it three arrays of TableColumn objects: the list of all available fields, the currently selected ones and the one chosen to go in the accordion. You also need to subscribe the event columnsSelectionChanged, where you will need to update the arrays and perhaps save them to the server or to the local storage. Finally, to handle resizing, you will need to handle the columnResized event to set the new size.

The type TableColumn allows you to specify the label, the column name (for sorting), the field name and the rendering if you can live with the defaults or a custom template if you need to render your field in some special way. In the example, there are two custom template for the deal ID and the asset class.

Note that the component lives in a separate angular module, to help with tree shaking (saving space in your app by excluding items you do not use). To use this component, therefore, you need to import the module GrootTableAutocolModule.
