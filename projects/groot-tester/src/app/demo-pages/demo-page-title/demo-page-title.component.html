Page titles

Groot style

Since showing a title of the page is rather common, you can use the groot component groot-page-title. You can see a demo above. Simply add it to your main component (AppComponent) and then configure your router as follows: 

{
  component: DemoPageTitleComponent,
  path: 'demo/page-title',
  data: {pageTitle: {title: 'Page Title Demo', subTitle: 'Page sub title'}}},
},

Darwin (XDCE) Style

We also have a different version of the page title component that looks like the XDCE one. This version doesn't use the router, but rather has to be included manually in all the actual pages. The usage is as follow: 

<groot-darwin-page-title
  icon="fa fa-star"
  title="This is the page title"
  subtitle="Subtitle of your page"
></groot-darwin-page-title>
