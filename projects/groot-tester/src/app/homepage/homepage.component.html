GRooT

Internal framework for Angular apps

GRooT is our internal library of angular components and styles, designed to provide the commonly used components that you need in an enterprise web app.

This documentation is available at: http://groot-lib-groot.192.168.232.45.nip.io/groot-doc

Getting started

Use the menu bar you can access various demo pages which showcase the various GRooT components.

You might want to start with our Getting Started guide.
