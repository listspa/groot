GRooT

Internal framework for Angular apps

GRooT is our internal library of angular components and styles, designed to provide the commonly used components that you need in an enterprise web app.

This documentation is available at: http://192.168.232.41/groot-doc

Library content

With the menu bar you can access various demo pages which showcase the various GRooT components.
