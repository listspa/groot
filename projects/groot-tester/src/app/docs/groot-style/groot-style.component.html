Stylesheets

Groot is based on Bootstrap v5.3. This means that you can use the standard Bootstrap grid and components you know.

Groot customizes Bootstrap's style by using its variables mechanism; for instance, we redefine the color primary or the fonts. However, you can override Groot's variables yourself and create a "skin".

As an example, say you want to override the primary color to be the brand color for a customer. At the begging of your style.scss file, before including Groot, simply define the variable: 

$primary: #258900

With this definition the nav bar, the highlight of form fields, and so on will respect your new color.

There are too many variables to document; please refer to the source file groot-variables.scss for details.
