Icons

Groot v4 uses Font Awesome 6.4 for the icon set. Please refer to their icon search page.

To use an icon, simply use <i class="fa-YYY fa-XXX"></fa>, where YYY is the style (solid, regular, brand) of the icon and XXX is the icon name you have found on FontAwesome's site.

Do you still want Font Awesome 4?

Is it possible to make the two Font Awesome versions coexist, you just have to override fonts and the core Font Awesome version 6 scss with the respective version 4 references, just override them after the main Groot scss import @import "@listgroup/groot/src/style/groot";.

To give you an example in your application: 

@import "@listgroup/groot/src/style/groot";

// Override Font Awesome v6 with v4
$fa-font-path: "~font-awesome/fonts";
@import "font-awesome/scss/font-awesome";
